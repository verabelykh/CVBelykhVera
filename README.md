# Portfolio (из Figma)

## Как запустить локально
1. Установи Node.js (https://nodejs.org, версия LTS)
2. В папке проекта выполни:
   npm install
   npm run dev
3. Открой ссылку, которую покажет терминал (обычно http://localhost:5173)

## Важно про картинки
Все изображения сейчас подключены по временным ссылкам Figma (действуют 7 дней с момента экспорта — с 8 июля 2026).
Перед долгосрочным использованием их нужно скачать и положить в папку `src/assets`, заменив ссылки в `src/Frame2136141531.jsx` на локальные пути, например:
    import img1 from './assets/img1.png'
